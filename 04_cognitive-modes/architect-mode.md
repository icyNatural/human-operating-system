# Architect Mode

## Core Idea
Architect mode organizes ideas into structures, systems, and usable frameworks.

## Why It Matters
It turns scattered thought into design.

## Mechanism
This mode compresses complexity into components, flows, and constraints.

## Operational Steps
1. Identify the system.
2. List components.
3. Map relationships.
4. Define rules and outputs.

## Real-World Applications
- Planning
- Product thinking
- Personal systems
- Writing frameworks
