# Socratic Mode

## Core Idea
Socratic mode uses questions to expose assumptions, contradictions, and unclear definitions.

## Why It Matters
It sharpens thought and removes hidden confusion.

## Mechanism
Question sequences destabilize weak reasoning and refine concepts.

## Operational Steps
1. Define the claim.
2. Ask what is assumed.
3. Ask what follows if true.
4. Search for contradiction.

## Real-World Applications
- Study
- Philosophy
- Debugging beliefs
- Better conversations
