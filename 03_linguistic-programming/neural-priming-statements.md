# Neural Priming Statements

## Core Idea
Certain statements prepare the mind for a chosen mode before action.

## Why It Matters
Preparation changes the probability of the next behavior.

## Mechanism
Language influences attention, expectation, emotion, and motor readiness.

## Operational Steps
1. Pick a target state.
2. Write one sentence that installs it.
3. Repeat it before action.
4. Measure whether it changes behavior.

## Real-World Applications
- Work starts
- Gym sessions
- Social entry
- Study blocks

## Observations / Experimental Notes
Save the statements that consistently work.
